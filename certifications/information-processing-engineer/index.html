기출문제 및 연습 문제

진행: 0/15 정답률: 0%

다음은 정보처리기사 자격증 준비를 위한 기출문제 및 연습 문제입니다. 각 문제에 답을 선택하고 정답을 확인할 수 있습니다. 각 문제는 관련 포스팅과 연결되어 있어 상세한 설명을 확인할 수 있습니다.

기출문제 리스트

번호 제목 주제 CBT 바로보기 문제다운 해설다운 문제+해설

2024년 기출문제

문제 1 소프트웨어 설계

Q: 요구사항 분석 단계에서 사용자 요구사항을 명확히 파악하기 위한 기법이 아닌 것은?

정답: D

해설: 코드 리뷰는 구현 단계에서 수행하는 활동이며, 요구사항 분석 단계의 기법이 아닙니다.

문제 2 데이터베이스 구축

Q: 관계형 데이터베이스에서 정규화의 목적은?

정답: A

해설: 정규화의 주요 목적은 데이터 중복을 제거하고 데이터 무결성을 향상시키는 것입니다.

문제 3 소프트웨어 개발

Q: 소프트웨어 개발 생명주기(SDLC)에서 테스트 단계의 목적은?

정답: B

해설: 테스트 단계는 소프트웨어의 오류를 발견하고 수정하는 것이 목적입니다.

문제 4 알고리즘

Q: 정렬 알고리즘 중 시간 복잡도가 O(n log n)인 것은?

정답: C

해설: 병합 정렬은 시간 복잡도가 O(n log n)인 효율적인 정렬 알고리즘입니다.

문제 5 SQL

Q: SQL에서 여러 테이블의 데이터를 결합하여 조회할 때 사용하는 구문은?

정답: A

해설: JOIN은 여러 테이블의 데이터를 결합하여 조회할 때 사용하는 SQL 구문입니다.

2025년 기출문제

문제 6 운영체제

Q: 프로세스 스케줄링 알고리즘 중 라운드 로빈(Round Robin)의 특징은?

정답: B

해설: 라운드 로빈은 시간 할당량을 두고 프로세스를 순환 방식으로 실행하는 스케줄링 알고리즘입니다.

문제 7 네트워크

Q: OSI 7계층 모델에서 전송 계층(Transport Layer)의 주요 기능은?

정답: C

해설: 전송 계층은 신뢰성 있는 데이터 전송과 흐름 제어를 담당합니다.

문제 8 보안

Q: XSS(Cross-Site Scripting) 공격을 방어하기 위한 가장 효과적인 방법은?

정답: A

해설: XSS 공격을 방어하기 위해서는 입력값을 검증하고 출력값을 이스케이프 처리해야 합니다. 통합 보안 취약점 점검 및 ISMS-P 인증 대응 실무 포스트에서 자세한 설명을 확인하세요.

문제 9 자료구조

Q: 스택(Stack) 자료구조의 특징이 아닌 것은?

정답: D

해설: 스택은 LIFO 구조이며, FIFO는 큐(Queue)의 특징입니다.

문제 10 데이터베이스

Q: 트랜잭션의 ACID 특성 중 일관성(Consistency)의 의미는?

정답: B

해설: 일관성은 트랜잭션 전후 데이터베이스가 일관된 상태를 유지하는 것을 의미합니다.

2026년 기출문제

문제 11 프로그래밍 언어

Q: 객체지향 프로그래밍의 주요 특징이 아닌 것은?

정답: C

해설: 객체지향 프로그래밍의 주요 특징은 캡슐화, 상속, 다형성이며, 순차 실행만 가능한 것은 특징이 아닙니다.

문제 12 시스템 분석

Q: 시스템 분석 단계에서 사용하는 다이어그램이 아닌 것은?

정답: A

해설: 코드 다이어그램은 시스템 분석 단계에서 사용하는 다이어그램이 아닙니다. DFD, 유스케이스, ER 다이어그램은 분석 단계에서 사용됩니다.

문제 13 소프트웨어 공학

Q: 소프트웨어 개발 방법론 중 애자일(Agile) 방법론의 특징은?

정답: D

해설: 애자일 방법론은 반복적 개발과 고객과의 지속적인 협업을 특징으로 합니다.

문제 14 데이터베이스 설계

Q: 데이터베이스에서 인덱스(Index)를 사용하는 주요 목적은?

정답: B

해설: 인덱스는 데이터 검색 성능을 향상시키기 위해 사용됩니다.

문제 15 정보시스템 구축 관리

Q: 프로젝트 관리에서 위험 관리의 주요 단계가 아닌 것은?

정답: C

해설: 위험 관리는 위험을 식별, 분석하고 대응 계획을 수립하는 것이지, 무시하는 것이 아닙니다.
