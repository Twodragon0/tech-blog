다음은 AWS SAA 자격증 준비를 위한 기출문제 및 연습 문제입니다. 각 문제에 답을 선택하고 정답을 확인할 수 있습니다. 각 문제는 관련 포스팅과 연결되어 있어 상세한 설명을 확인할 수 있습니다.

기출문제 리스트

번호 제목 주제 CBT 바로보기 문제다운 해설다운 문제+해설

2024년 이후 기출문제

문제 1 VPC 및 네트워크 보안

Q: 여러 가용 영역에 걸쳐 고가용성 웹 애플리케이션을 배포하려고 합니다. 인터넷에서의 트래픽을 분산하고, 프라이빗 서브넷의 애플리케이션 서버에 안전하게 연결하려면 어떤 아키텍처를 사용해야 합니까?

정답: A

해설: Application Load Balancer를 퍼블릭 서브넷에 배치하면 인터넷 트래픽을 받을 수 있고, 프라이빗 서브넷의 애플리케이션 서버는 보안이 강화됩니다. VPC 보안 아키텍처 포스트에서 자세한 설명을 확인하세요.

문제 2 IAM 및 접근 제어

Q: 여러 AWS 계정을 관리하는 조직에서 중앙 집중식 거버넌스를 구현하려고 합니다. 모든 계정에서 특정 서비스 사용을 제한하려면 어떤 서비스를 사용해야 합니까?

정답: B

해설: AWS Control Tower 및 SCP 포스트에서 자세한 설명을 확인하세요.

문제 3 데이터베이스 보안

Q: 프라이빗 서브넷의 RDS 데이터베이스에 안전하게 접근하기 위한 게이트웨이를 구축하려고 합니다. 인터넷에서 직접 접근을 차단하면서도 애플리케이션 서버에서 접근할 수 있도록 하려면 어떤 아키텍처를 사용해야 합니까?

정답: A

해설: 데이터베이스 접근 게이트웨이 포스트에서 자세한 설명을 확인하세요.

문제 4 비용 최적화

Q: 예측 가능한 워크로드를 위해 EC2 인스턴스 비용을 최적화하려고 합니다. 가장 비용 효율적인 옵션은 무엇입니까?

정답: B

해설: AWS FinOps 포스트에서 비용 최적화 전략을 확인하세요.

문제 5 S3 및 스토리지

Q: 자주 접근하지 않는 데이터를 비용 효율적으로 저장하려고 합니다. 데이터는 즉시 접근 가능해야 하지만, 자주 사용되지 않습니다. 어떤 S3 스토리지 클래스를 사용해야 합니까?

정답: C

해설: S3 Standard-IA는 자주 접근하지 않는 데이터를 즉시 접근 가능한 상태로 저장하면서도 비용을 절감할 수 있는 스토리지 클래스입니다. S3 Glacier Instant Retrieval도 가능하지만, Standard-IA가 더 일반적인 선택입니다.

문제 6 고가용성 및 확장성

Q: 트래픽이 예측 불가능하게 변동하는 웹 애플리케이션을 운영하고 있습니다. 트래픽 증가에 따라 자동으로 인스턴스를 추가하고, 트래픽 감소 시 인스턴스를 제거하려면 어떤 서비스를 사용해야 합니까?

정답: B

해설: Auto Scaling Group은 CloudWatch 메트릭을 기반으로 자동으로 인스턴스를 추가하거나 제거합니다. Elastic Load Balancer와 함께 사용하면 트래픽을 여러 인스턴스에 분산시킬 수 있습니다.

문제 7 데이터베이스

Q: 읽기 작업이 많은 애플리케이션에서 RDS 데이터베이스의 성능을 향상시키려고 합니다. 읽기 전용 복제본을 여러 개 생성하여 읽기 트래픽을 분산시키려면 어떤 방법을 사용해야 합니까?

정답: A

해설: RDS Read Replica는 읽기 전용 복제본으로, 여러 개 생성하여 애플리케이션에서 직접 연결하여 읽기 트래픽을 분산시킬 수 있습니다. Load Balancer는 RDS에 직접 연결할 수 없습니다.

문제 8 네트워킹

Q: 프라이빗 서브넷의 EC2 인스턴스가 인터넷에 접근하여 소프트웨어 업데이트를 다운로드해야 합니다. 인터넷 게이트웨이를 사용할 수 없는 프라이빗 서브넷에서 인터넷 접근을 제공하려면 어떤 서비스를 사용해야 합니까?

정답: A

해설: NAT Gateway는 프라이빗 서브넷의 인스턴스가 인터넷에 접근할 수 있도록 하는 관리형 서비스입니다. NAT Instance도 가능하지만, NAT Gateway가 더 안정적이고 관리가 쉽습니다.

문제 9 Lambda 및 서버리스

Q: S3 버킷에 파일이 업로드될 때마다 자동으로 이미지를 리사이징하는 서버리스 애플리케이션을 구축하려고 합니다. 어떤 서비스 조합을 사용해야 합니까?

정답: A

해설: S3 Event Notification을 사용하여 파일 업로드 이벤트를 Lambda 함수로 전달하고, Lambda 함수에서 이미지를 리사이징한 후 다시 S3에 저장하는 것이 가장 효율적인 방법입니다.

문제 10 보안 및 암호화

Q: S3 버킷에 저장된 민감한 데이터를 암호화하려고 합니다. 서버 측 암호화(SSE)를 사용하면서도 암호화 키를 완전히 제어하려면 어떤 방법을 사용해야 합니까?

정답: C

해설: SSE-C(Server-Side Encryption with Customer-Provided Keys)는 고객이 제공한 암호화 키를 사용하여 S3에서 데이터를 암호화합니다. 키를 완전히 제어할 수 있지만, 키 관리 책임도 고객에게 있습니다.

문제 11 CloudFront 및 CDN

Q: 전 세계 사용자에게 정적 웹사이트 콘텐츠를 빠르게 제공하려고 합니다. S3 버킷에 저장된 콘텐츠를 전 세계 엣지 로케이션을 통해 제공하려면 어떤 서비스를 사용해야 합니까?

정답: B

해설: CloudFront는 AWS의 CDN(Content Delivery Network) 서비스로, S3 버킷을 오리진으로 사용하여 전 세계 엣지 로케이션을 통해 콘텐츠를 빠르게 제공할 수 있습니다.

문제 12 모니터링 및 로깅

Q: EC2 인스턴스의 CPU 사용률이 80%를 초과할 때 알림을 받고 싶습니다. 어떤 서비스를 사용해야 합니까?

정답: A

해설: CloudWatch Alarms를 사용하여 CPU 사용률 메트릭을 모니터링하고, 임계값을 초과하면 SNS(Simple Notification Service)를 통해 알림을 전송할 수 있습니다.

문제 13 비용 최적화

Q: 예측 불가능한 워크로드를 실행하려고 합니다. 중단이 허용되는 배치 작업에 가장 적합한 EC2 인스턴스 유형은 무엇입니까?

정답: C

해설: Spot Instances는 중단이 허용되는 워크로드에 가장 비용 효율적인 옵션입니다. 최대 90%까지 할인된 가격으로 제공되지만, AWS가 용량이 필요할 때 인스턴스를 중단할 수 있습니다.

문제 14 데이터베이스 고가용성

Q: RDS 데이터베이스의 고가용성을 보장하려고 합니다. 자동 장애 조치를 제공하면서도 동기식 복제를 사용하려면 어떤 구성을 사용해야 합니까?

정답: B

해설: RDS Multi-AZ는 다른 가용 영역에 동기식 복제본을 생성하고, 자동으로 장애 조치를 수행합니다. Read Replica는 비동기식 복제이며 자동 장애 조치를 제공하지 않습니다.

문제 15 네트워크 보안

Q: VPC 내의 EC2 인스턴스 간 트래픽을 제어하려고 합니다. 특정 IP 주소에서만 접근을 허용하려면 어떤 보안 메커니즘을 사용해야 합니까?

정답: C

해설: Security Group은 인스턴스 레벨에서 트래픽을 제어하고, Network ACL은 서브넷 레벨에서 트래픽을 제어합니다. Defense in Depth 전략을 위해 둘 다 사용하는 것이 권장됩니다.

학습 전략 시험 준비

효과적인 학습 방법:

  • NxtCloud 블로그 활용: 문제 풀이 사고 과정을 익히는 데 매우 유용합니다. 하루에 2-3개씩 꾸준히 읽어보세요.
  • 빠른 개념 정리: 개념을 완벽하게 정리하기보다는 빠르게 훑고, 문제 풀이 과정에서 부족한 개념을 채우는 전략이 효율적입니다.
  • 준비 기간: 가능하면 한 달 이내로 시험을 보는 것을 권장합니다. 준비 기간이 길어지면 개념과 문제 풀이 감각이 휘발되기 쉽습니다.
시험 팁 시험 당일

시험 당일 주의사항:

  • 추가 시간 신청: 시험 신청 전에 40분 추가 시간을 요청할 수 있습니다. 가능하면 신청하세요.
  • 신분증 준비: 주민등록증은 영문 이름이 없어 사용할 수 없습니다. 여권 또는 국제 운전면허증을 준비하세요.
  • 덤프 문제 기대하지 않기: 덤프 문제가 많이 나올 것이라는 가정은 하지 마세요. 실제 시험에서는 덤프에서 본 적 없는 문제가 많이 나올 수 있습니다.
  • 처음 보는 서비스 대비: iSCSI처럼 처음 보는 서비스가 등장할 수 있으므로 당황하지 마세요.
  • 시험 결과: 시험 후 몇 시간 내에 합격/불합격 결과를 이메일로 받을 수 있습니다.
