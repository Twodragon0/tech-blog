# Optimize Scripts

## Steps
1. Identify target scripts
2. Check for common issues: error handling, rate limiting, logging
3. Review dependencies and imports
4. Check for hardcoded values that should be config
5. Verify scripts follow project conventions
6. Run linting: `ruff check` or `flake8` if available
7. Test optimized scripts
