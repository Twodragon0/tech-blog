# Notion Sync

## Steps
1. Use Notion MCP to query relevant pages/databases
2. Extract content and format as Jekyll markdown
3. Handle images and media assets
4. Create/update blog posts with proper frontmatter
5. Verify Jekyll build after sync
6. Update tracking (last sync date, synced pages)
