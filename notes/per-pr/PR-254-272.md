## 학습 포인트 (재사용 가능한 패턴)

### 1. Liquid include quote 안전 패턴 (PR #268, #270, #271, #272)
**문제**: Liquid include tag 의 string argument 에 quote 를 포함하면 Jekyll 파싱 실패 또는 unintended substitution.

**해결책**:
- Outer quote: double `"` 로 고정
- Inner quote 및 특수 문자 escape:
  - `'` → `&#x27;`
  - `"` → `&quot;`
  - `&` → `&amp;`

**구현**:
```python
def _html_escape_quotes(text):
  """Escape HTML entities for Liquid include tag arguments."""
  if not text:
    return text
  return text.replace('&', '&amp;') \
             .replace('"', '&quot;') \
             .replace("'", '&#x27;')
```

**테스트 전략**: `_html_escape_quotes()` 헬퍼 직접 단위 테스트 + 통합 테스트 (실제 포스트 레이아웃 렌더링).
