{%- comment -%} Self-hosted Noto Sans KR — 2-tier woff2 subset. Tier-1 (this @font-face): top ~1k Hangul syllables + Latin + Jamo + symbols (~200 KB / weight, eager preload). Tier-2 (lazy): remaining ~10k Hangul syllables (~480-500 KB / weight, injected by head-runtime.js after idle). Both tiers claim the same unicode-range U+AC00-U+D7A3 so the browser uses whichever face has the requested glyph. See scripts/build/generate_noto_2tier_subset.py for the source pipeline. {%- endcomment -%}
