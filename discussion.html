GitHub Discussions를 통해 커뮤니티와 소통합니다. 기술적인 질문, 피드백, 또는 주제 제안 등 자유롭게 참여해 주세요!

Discussion Categories

General

일반적인 대화와 커뮤니티 소통

Q&A

기술적인 질문과 답변

Ideas

새로운 아이디어와 제안

Show and Tell

프로젝트와 성과 공유

How to Participate

  1. GitHub 계정으로 로그인합니다
  2. Discussions 페이지를 방문합니다
  3. 적절한 카테고리를 선택하여 새 Discussion을 시작합니다
  4. 각 블로그 포스트 하단의 댓글로도 참여할 수 있습니다

Community Guidelines

  • 존중: 다른 참여자를 존중하고 건설적인 대화를 유지해 주세요
  • 명확성: 질문이나 의견을 명확하게 작성해 주세요
  • 관련성: 보안, DevSecOps, 클라우드 관련 주제에 집중해 주세요
  • 공유: 유용한 정보와 경험을 자유롭게 공유해 주세요
